# 命名方案候选清单

## 一、站点副标题（替代"宝藏之地"）

| 编号 | 方案 | 风格 | 说明 |
|------|------|------|------|
| A1 | **Digital Garden** | 经典英文 | 最正统的"数字花园"，技术圈广泛认可的概念 |
| A2 | **The Garden** | 极简英文 | 去掉 Digital，更留白，更诗意 |
| A3 | **数字花园** | 中文直译 | 直白但优雅，国内受众友好 |
| A4 | **拾光庭** | 中式雅致 | "拾取时光之庭院"，有收集和沉淀的意象 |
| A5 | **思迹** | 二字凝练 | "思考的痕迹"，极简但有深度 |
| A6 | **栖码处** | 技术诗意 | "代码栖息之处"，software student 的归属感 |
| A7 | **星丛笔记** | 文学感 | 引用本雅明"星丛"概念，碎片化的知识如繁星 |
| A8 | **幽篁里** | 古诗引用 | 王维"独坐幽篁里"，极雅但可能过于古典 |
| A9 | **未命名花园** | 留白感 | 刻意不命名，有种"正在生长中"的状态 |
| A10 | **纸间电波** | 复古数字 | 纸（文字）+ 电波（数字），有穿越时空感 |

**推荐**: A1（Digital Garden）或 A4（拾光庭）或 A6（栖码处）
